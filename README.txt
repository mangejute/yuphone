LifeTalk mobile PWA release for GitHub Pages.

Upload the full contents of this folder to a GitHub repository root or /docs folder.
Then enable GitHub Pages for that branch and folder.
This build is installable as a phone PWA.
The 241MB local SenseVoice package is intentionally excluded because it exceeds normal GitHub file limits.
The release defaults to browser speech features when available.
Keep .nojekyll, index.html, 404.html, manifest.webmanifest, sw.js, pwa-icon.svg, pwa-icon-192.png, and pwa-icon-512.png together.
